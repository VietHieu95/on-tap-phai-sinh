# Ôn tập phái sinh tài chính

Web học và luyện đề phái sinh tài chính bằng tiếng Việt.

## Nội dung

- 15 chương theo bộ đề gốc.
- Lý thuyết tiếng Việt trước mỗi chương.
- 460 câu hỏi luyện tập, có đáp án và lời giải.
- Chế độ luyện từng câu, thi thử theo chương và ôn câu sai.
- Tiến độ lưu trong trình duyệt bằng `localStorage`.

## Chạy local

```bash
npm install
npm run dev
```

Mở `http://127.0.0.1:3000`.

## Kiểm tra

```bash
npm run lint
npm run build
```

## Deploy Vercel qua GitHub

1. Tạo GitHub repo mới.
2. Push repo này lên GitHub.
3. Vào Vercel, chọn **Add New Project**, import repo GitHub vừa tạo.
4. Giữ mặc định framework Next.js, deploy.
